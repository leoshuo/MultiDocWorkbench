# 快速参考（前台用户端）

Document Workspace 1.6.3

适用：多文档处理工作台（前台用户端）

## 快速步骤

1. **上传文档**：点击来源面板的 + 按钮上传文件
2. **选择写作按钮**：在对话区选择预配置的写作按钮
3. **查看结果**：在对话记录中查看执行结果

## 默认按钮

| 按钮名称 | 功能说明 |
|---------|---------|
| 日报合并写作（主任版） | 合并多份日报生成汇总 |
| 竞品分析报告写作 | 生成竞品分析报告 |
| 自定义写作 | 自定义写作任务 |

## 界面说明

### 来源面板（左侧）
- 显示已上传的文档列表
- 支持多文件批量上传
- 点击文档可选中/取消选中

### 对话面板（中间）
- 显示AI对话记录
- 提供写作按钮快捷入口
- 支持自由对话输入

### Studio面板（右侧）
- 笔记管理
- 输出内容保存

## 常用操作

### 上传文档
1. 点击来源面板右上角 + 按钮
2. 选择本地文件（支持 .txt, .md, .docx）
3. 等待上传完成

### 清除文档
1. 点击来源面板右上角垃圾桶图标
2. 确认清除所有文档

### 切换工作台
1. 点击页面顶部"切换后台管理工作台"
2. 进入经验沉淀工作台

## 说明

- 任一工作台上传文档会同步到另一端
- 失败或 pass 的 Section 会在对话中给出原因
- 切换工作台不会清空数据
- 未配置 QWEN_API_KEY 时，AI 回复会使用占位结果或规则抽取
